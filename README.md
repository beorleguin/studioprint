# Studio Print Web

Landing one page en Next.js + CSS.

## Ejecutar

```bash
npm install
npm run dev
```

Abrir:

```txt
http://localhost:3000
```

## Imágenes usadas

## Nota visual

El fondo de todo el sitio está unificado desde `body::before` y `body::after`.
No hay fondos por sección, por eso navbar, hero, servicios, trabajos, proceso y contacto comparten el mismo fondo con partículas.
